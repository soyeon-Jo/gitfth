## 파일 관리
### 1.  편집
 vim '파일이름' 
 ```java
 vim hello
 ```
##### vim 안에 들어간 상태에서 i 버튼 누르면 편집(insert) 가능, 
##### esc 누르면 편집 종료
##### esc 누른 상태에서 <u>:wq</U> 누르면 저장하고 나가기  </u>  
  
### 2. 상태 확인
#### : 파일의 상태 확인 가능  
cat '파일이름' 
```java
cat hello
```
### 3. git 상태 확인
####  git 의 현 상황 알려줌  
```java
git status
```

### 4. 파일 추가
```java
git add hello
```
git은 버전관리를 하기 위해서 등록을 해줘야하므로  파일을



 추가한다.
